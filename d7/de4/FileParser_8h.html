Converts a vector of any type to a vector of strings.

This function template takes a vector of any type and converts each element to a string. The conversion is done using a stringstream and the scientific notation is used for the conversion. The precision of the conversion can be specified as an optional parameter.

precisionThe precision to be used for the conversion. This is optional and defaults to 19.
Returns
A vector of strings where each string is the string representation of the corresponding element in the input vector.

Example usage:

std::vector<std::string> stringVector = asStringVector(intVector);

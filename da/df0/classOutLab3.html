Detailed Description

A class for performing numerical integration using Composite Newton-Cotes Formulas.

This class is derived from the Project class and implements the numerical integration using Composite Newton-Cotes Formulas. It takes command line arguments, parses the inputs, performs the integration, and outputs the results.

Detailed Description

Struct to hold variables for Taylor Series.

Struct representing the output vector for the BLAS library.

A structure to hold the output of the relaxation method.

Struct representing the output parameters for the Newton-Cotes integration methods.

This structure contains the input parameters, the solution, and the execution time.

# Locating Source Files
Sources and file paths for each deliverable follow the following structure:
```
├── src  <= You are now here
│   ├── project              <= Class project deliverables
│   │   ├── project1
│   │   ├── ...
│   │   ├── project5
│   ├── labs                 <= InLab, OutLab deliverables
│   │   ├── inlab1
│   │   ├── outlab1
│   │   ├── ...
│   │   ├── inlab13
│   │   ├── outlab13
│   ├── ...
├── ...
```

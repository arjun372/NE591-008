This structure is used to map tones using a scaling factor and a growth rate. It is a template structure, so it can be used with any data type.

Data Fields
T growth_rate The growth rate for the tone map.

This is the rate at which the tone map grows. It is initialized to a moderate value (0.3) to allow for a reasonable growth rate.

T scaling_factor The scaling factor for the tone map.

This is the factor by which the input tone will be scaled. It is initialized to a very small value (1e-7) to prevent large initial tones.

Detailed Description

template<typename FloatType>
struct MyPhysics::Diffusion::Params< FloatType >

Template struct to hold parameters for the diffusion solver.

This struct contains parameters necessary for solving diffusion problems.

Calculates the mesh spacings in the given direction.

Computes the mesh spacings using the equation: spacing = length / (points + 1). These spacings are used to discretize the diffusion operator in the given equation.

pointsThe number of mesh points in the given direction, excluding the boundary points.

Computes the sum of a series from a start to a stop index.

This function computes the sum of a series from a start to a stop index. The series is defined as the square of the ratio of (1.0f + log10(i-1)) to ((i-1) + log10(i)) for each i from start to stop. If start is greater than stop, the function swaps the two values.

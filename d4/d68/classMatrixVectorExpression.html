Detailed Description

template<template< typename > class MatrixType, template< typename > class VectorType, typename T>
class MatrixVectorExpression< MatrixType, VectorType, T >

Represents an expression for the element-wise multiplication of a matrix and a vector.

This class defines an expression for the element-wise multiplication of a matrix and a vector. It can be used to perform element-wise multiplication without explicitly creating a new matrix.

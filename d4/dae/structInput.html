Detailed Description

Struct representing the input parameters for the Newton-Cotes integration methods.

Struct representing the input matrices for the BLAS library.

A structure to hold the input parameters.

A structure to hold the input parameters for the relaxation method.

This structure contains the convergence threshold, maximum number of iterations, size of the matrices, coefficient matrix, vector of constants, and the set of methods.

Parameters
kA long double representing the scalar multiplier for [A].
MA size_t representing the row rank for [A] and [B].
NA size_t representing the column rank for [B] and row rank for [F].
JA size_t representing the column rank for [F].

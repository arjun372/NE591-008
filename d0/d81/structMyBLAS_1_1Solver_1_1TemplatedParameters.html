Detailed Description

template<template< typename > class MatrixType, template< typename > class VectorType, typename T>
struct MyBLAS::Solver::TemplatedParameters< MatrixType, VectorType, T >

Template struct to hold parameters for a solver with templated matrix and vector types.

This struct contains parameters necessary for solving problems with templated matrix and vector types.

Template Parameters
MatrixTypeThe matrix type used for storing coefficients.
VectorTypeThe vector type used for storing constants.
TThe data type used for storing parameters.

Detailed Description

template<typename InputType, typename CommandLineParserType, typename OutputType>
class MPIProject< InputType, CommandLineParserType, OutputType >

This class is a template for creating MPI projects. It provides methods for initializing MPI, getting inputs, executing the project, and finalizing MPI.

Template Parameters
InputTypeThe type of the input data.
CommandLineParserTypeThe type of the command line parser.
OutputTypeThe type of the output data.

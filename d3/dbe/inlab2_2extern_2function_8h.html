Fill the fx vector, optionally using the values in the x vector.

Parameters
xThe input vector containing x values.
fxThe output vector to be filled with fx values.

This function takes an input vector x and an output vector fx. It fills the fx vector with values. The fx vector is assumed to be initially zeroed out. The function does not return any value.

Note
The x and fx vectors must have the same size.
The fx vector will be modified by this function.
The x vector will not be modified by this function.

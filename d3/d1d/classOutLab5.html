Detailed Description

This class is a child of the Project class and is used to solve a system of linear equations using forward and back substitution.

The class takes in command line arguments and uses them to solve the system of equations.

This function runs the project.

It solves the system of linear equations using forward and back substitution.

Parameters
outputsThe output vector
inputsThe input matrices
valuesThe variable map
  1. Given the matrix A = LU and vector b, solve Ax = b, which is LUx = b.
  2. Perform row pivoting on A to obtain a permuted matrix PA = LU, where P is a permutation matrix.
  3. Let y = Ux. Now, we have two systems of linear equations: Ly = Pb and Ux = y.
  4. Solve the first system Ly = Pb using forward substitution.
  5. Solve the second system Ux = y using backward substitution.

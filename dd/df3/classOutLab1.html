This function runs the main logic of the program.

Parameters
valuesA map of variables parsed from the command line arguments.

The function first extracts the input parameters from the values map. It then builds and prints three matrices A, B, and F using the input parameters. After that, it computes and prints three more matrices C, D, and E using the previously built matrices and the scalar value from the input parameters.
